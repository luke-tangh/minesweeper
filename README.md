# minesweeper
## Brief
A C++ version of a classic game minesweeper.

## Environment
- `c++ 17`
- `EasyX`

## Shortcut keys
- `LEFTCLICK` | `RIGHTCLICK` | `MIDDLECLICK`
- `ESC` - exit 
- `R` - restart
